# Electricity Bill Calculator

เว็บคำนวณค่าไฟฟ้าอย่างง่ายด้วย HTML, CSS และ JavaScript

## ไฟล์ในโปรเจกต์

- `index.html` โครงสร้างหน้าเว็บและฟอร์มกรอกข้อมูล
- `style.css` ไฟล์ตกแต่งหน้าตาเว็บ
- `script.js` โค้ดคำนวณค่าไฟฟ้าและแสดงผล

## วิธีใช้งาน

1. เปิดไฟล์ `index.html` ด้วยเว็บเบราว์เซอร์
2. กรอกจำนวนหน่วยไฟฟ้าที่ใช้เป็น kWh
3. ปรับค่า Ft, ค่าบริการ หรือ VAT ได้ตามต้องการ
4. กดปุ่ม `คำนวณ` หรือพิมพ์ค่าในช่อง ระบบจะแสดงผลให้อัตโนมัติ

## สิ่งที่ระบบคำนวณ

- ค่าไฟฐานตามอัตราขั้นบันได
- ค่า Ft ตามจำนวนหน่วยที่ใช้
- ค่าบริการรายเดือน
- VAT
- ยอดรวมที่ต้องชำระโดยประมาณ
- ตารางแจกแจงค่าไฟแต่ละช่วงหน่วย

## หมายเหตุ

อัตราค่าไฟที่ใช้ในตัวอย่างเป็นอัตราสำหรับทำแบบฝึกหัด/โปรเจกต์เว็บเบื้องต้น
หากต้องการใช้กับบิลจริง ควรตรวจสอบอัตราล่าสุดจากหน่วยงานการไฟฟ้าอีกครั้ง
